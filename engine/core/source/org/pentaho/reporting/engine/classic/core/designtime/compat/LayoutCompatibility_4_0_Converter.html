Layout-Property migration from Pentaho Reporting 3.9.1 or older to Pentaho Reporting 4.0.0

Before you can use any existing report with the new crosstab-elements or any new layout capabilities you must migrate the reports to the Pentaho Reporting 4.0 format. This migration corrects invalid properties that were not evaluated in older versions. The converter will reset these properties to their built-in default values.

Reports, groups and group-bodies that have a 'layout' style-property defined get reset to the default undefined state.
